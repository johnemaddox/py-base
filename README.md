## Fresh Start

This script will remove the existing git repo, create a new one, and remove fresh.sh.

```bash
./fresh.sh
```

# Python Base Project

This is a base project for quick prototyping in Python.

# Cleanup

The clean script removes all pycache directories.

```bash
./clean.sh
```

# Testing

Unit Tests can be executed through vscode or through terminal commands.

## Run All Tests

```bash
python -m unittest
```

## Run Specific Test
```bash
python -m tests.test_NAME
```
